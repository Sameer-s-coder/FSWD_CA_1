# Signup API

## How to Run
1. Install dependencies: `npm install`
2. Start server: `npm run dev`

## Endpoint
`POST /signup`

**Request Body:**
```json
{
  "username": "john",
  "email": "john@example.com",
  "password": "securepass",
  "dob": "2000-01-01"
}
